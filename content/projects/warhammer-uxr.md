## Overview

Served as a Game User Researcher at Games Workshop Group (May–Nov 2025), contributing to user research for the Warhammer game series.

## Responsibilities

- Assisted in designing user research plans for the Warhammer series, including questionnaire and semi-structured interview outlines
- Collaborated with the team to execute quantitative surveys and qualitative interviews in player communities, reaching 100+ target users
- Used SPSS for statistical analysis (reliability/validity testing, variance analysis)
- Conducted NVivo coding and thematic extraction on interview content
- Combined Python for data visualization, producing intuitive user behavior and needs insight reports
