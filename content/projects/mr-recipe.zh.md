## 概述

为探索MR技术在生活场景的落地价值，基于Unity引擎与Meta Quest 3设备开发MR烹饪指导原型，旨在通过虚实融合的交互方式解决传统烹饪教学中步骤不直观、操作易中断的痛点。

## 职责

- 完成MR场景搭建，将虚拟厨师形象与真实厨房环境进行空间叠加与定位校准
- 参与定性用户研究工作，通过访谈、观察等方式收集用户烹饪学习中的核心痛点

## 技术规格

| 组件 | 技术 |
|------|------|
| 引擎 | Unity |
| 设备 | Meta Quest 3 |
| 交互 | 空间锚定 + 手势追踪 |
| 研究 | 定性访谈 + 观察法 |

## 核心特性

- 虚拟厨师角色叠加在真实厨房空间中，完成空间校准
- 分步烹饪指导，不中断物理操作流程
- 基于定性研究发现驱动的迭代设计
